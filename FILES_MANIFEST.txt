AWS Preparation:
  □ Create AWS account
  □ Create IAM user with necessary permissions
  □ Install AWS CLI
  □ Install EB CLI
  □ Configure AWS credentials

AWS Deployment:
  □ Run: eb init -p python-3.11 calculator-app
  □ Run: eb create calculator-dev
  □ Run: eb deploy
  □ Open: eb open

Post-Deployment:
  □ Monitor: eb health
  □ View logs: eb logs --stream
  □ Set up CloudWatch alarms
  □ Configure custom domain (optional)
  □ Enable HTTPS/SSL (optional)
